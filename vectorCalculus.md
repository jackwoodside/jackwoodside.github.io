# Vector Calculus

## Coordinate System Conversion

### Spherical

$$x = r\sin\theta\cos\phi$$

$$y = r\sin\theta\sin\phi$$

$$z = r\cos\theta$$

$$\mathbf{\hat{x}} = \sin\theta\cos\phi\mathbf{\hat{r}} + \cos\theta\cos\phi\mathbf{\hat{\boldsymbol{\theta}}} - \sin\phi\mathbf{\hat{\boldsymbol{\phi}}}$$

$$\mathbf{\hat{y}} = \sin\theta\sin\phi\mathbf{\hat{r}} + \cos\theta\sin\phi\mathbf{\hat{\boldsymbol{\theta}}} + \cos\phi\mathbf{\hat{\boldsymbol{\phi}}}$$

$$\mathbf{\hat{z}} = \cos\theta\mathbf{\hat{r}} - \sin\theta\mathbf{\hat{\boldsymbol{\theta}}}$$

$$r = \sqrt{x^2 + y^2 + z^2}$$

$$\theta = \tan^{-1}\left(\sqrt{x^2 + y^2}/z\right)$$

$$\phi = \tan^{-1}(y/x)$$

$$\mathbf{\hat{r}} = \sin\theta\cos\phi\mathbf{\hat{x}} + \sin\theta\sin\phi\mathbf{\hat{y}} + \cos\theta\mathbf{\hat{z}}$$

$$\mathbf{\hat{\boldsymbol{\theta}}} = \cos\theta\cos\phi\mathbf{\hat{x}} + \cos\theta\sin\phi\mathbf{\hat{y}} - \sin\theta\mathbf{\hat{z}}$$

$$\mathbf{\hat{\boldsymbol{\phi}}} = -\sin\phi \mathbf{\hat{x}} + \cos\phi\mathbf{\hat{y}}$$

### Cylindrical

$$x = s\cos\phi$$

$$y = s\sin\phi$$

$$\mathbf{\hat{x}} = \cos\phi\mathbf{\hat{s}} - \sin\phi\mathbf{\hat{\boldsymbol{\phi}}}$$

$$\mathbf{\hat{y}} = \sin\phi\mathbf{\hat{s}} + \cos\phi\mathbf{\hat{\boldsymbol{\phi}}}$$

$$s = \sqrt{x^2 + y^2}$$

$$\phi = \tan^{-1}(y/x)$$

$$\mathbf{\hat{s}} = \cos\phi\mathbf{\hat{x}} + \sin\phi\mathbf{\hat{y}}$$

$$\mathbf{\hat{\boldsymbol{\phi}}} = -\sin\phi\mathbf{\hat{x}} + \cos\phi\mathbf{\hat{y}}$$
